Become a founder node

  1. Get a box with a public IPv4. Any small cloud VM (≈$3.5/mo) with a static IPv4 works — that's what makes a node reachable as a bootstrap entry point.
  2. Install & run epnd as a bootstrap node (sets up the service + 15-min auto-update):
    curl -fsSL https://github.com/50gramx/eapp-releases/releases/download/epnd-latest/install.sh | sh epnd serve --lite
  3. Announce your public IPv4 so IPv4-only peers can reach you — set EPN_ANNOUNCE_IP=<your.public.ip> in the service environment and restart.
  4. Enable status reporting — add the founder reporter (pushes your node's vitals + KPIs here, every 15 min) with EPND_STATUS_NODE_NAME and EPND_STATUS_NODE_CLASS=founder.
  5. Complete full founder mode — point a domain (e.g. node.yourname.com) at the box and set EPND_STATUS_DOMAIN. Full founders are listed by domain in the table above.

Full setup script + the exact reporter config are in epn-daemon/scripts/ (bootstrap-node-setup.sh, report-bootstrap-status.*).
