Founder / community program

What it takes to become visible here

  • Any epnd node can report Community and keeper nodes are welcome, not just bootstrap boxes — set `EPND_STATUS_NODE_CLASS=community` (the default) or `founder`.
  • Founder mode `founder` is reserved for a full, non-lite `epnd` bootstrap install — lite/mobile boxes self-report as `community`.
  • Status reporting Publish a unique `EPND_STATUS_NODE_NAME` so the page can track your node's own heartbeat history.
  • Full founder posture Attach a domain and announce it to show stable, domain-backed reachability.
