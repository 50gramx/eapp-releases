What it takes to become visible here

  • Reachable host Start with a machine that has a stable public IPv4 so it can act as a bootstrap entry point.
  • Founder mode Run `epnd` as a founder/bootstrap node and keep the auto-update path active.
  • Status reporting Publish `EPND_STATUS_NODE_NAME` and `EPND_STATUS_NODE_CLASS=founder` so the page can measure your node.
  • Full founder posture Attach a domain and announce it to show stable, domain-backed reachability.
